Feature Tour of My Location

This feature retrieves the user's location and performs geofencing.

Features:

  • 🗺️ Display of map
  • 📍 Pin user real-time location on map
  • ⭕ Click the 'Set Geofence' button to set a 200 meter radius geofence from the user's current location as the center

Status Indicators:

  • 🙆🏻‍♂️ Within geofence
  • 🙅🏻‍♂️ Outside of geofence
  • 🤷🏻‍♂️ No geofence is set
