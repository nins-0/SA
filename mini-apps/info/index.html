Feature Tour of My Location

This feature retrieves the user's location and performs geofencing.

Features:

  • 🗺️ Display of map
  • 📍 Pin user real-time location on map
  • ⭕ Click the + button for geofencing options

Geofencing Options:

  • Radius : 100m | 200m | 300m | 400m
  • Building : Building A | Building B

Status Indicators:

  • 🙆🏻‍♂️ Within geofence
  • 🙅🏻‍♂️ Outside of geofence
  • 🤷🏻‍♂️ No geofence is set
